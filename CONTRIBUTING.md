# Contributing to TourDesk

Thank you for your interest in contributing to TourDesk! We welcome all contributions, including bug reports, feature requests, documentation improvements, and pull requests.

## Code of Conduct

We are committed to fostering a welcoming and inclusive community. Please be respectful and supportive of all contributors.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bugfix (`git checkout -b feature/my-feature`).
3. Commit your changes using conventional commit messages.
4. Ensure the linter runs cleanly (`npm run lint`).
5. Submit a Merge Request (MR) on code.swecha.org.
